# 织旅 - 大学生时间规划器

一款专为大学生设计的时间管理工具，支持课程管理、任务规划、目标追踪和日记记录。

## 功能特点

- 📚 **课程管理**：表格化课表，支持1-18周、单双周课程，图片识别添加课程
- ✅ **任务清单**：任务截止日期管理，已完成任务归档
- 🎯 **目标管理**：大目标+小目标，子任务折叠展开，目标完成追踪
- 📝 **时光日记**：记录每天的心情与成长
- 🍅 **专注计时**：番茄钟专注计时，自动统计专注时长
